四级标题会将英文小写自动转为大写（css样式问题，已修复）

二、字体

  • 斜体
    要倾斜的文字左右分别用一个*号包起来
  • 加粗
    要加粗的文字左右分别用两个*号包起来
  • 斜体加粗
    要倾斜和加粗的文字左右分别用三个*号包起来
  • 删除线
    要加删除线的文字左右分别用两个~~号包起来
    示例：效果如下：
